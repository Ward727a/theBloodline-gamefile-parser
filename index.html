FYI:

  • The name of the properties is first, the type is second example:
    playerName - text
    here playerName is the name of the property and text is the type

  • If a type is surrounded with [[ ]] the it's a type from the game, example:
    PlayerItem - [[item]]

    The "item" type is not from Unreal Engine, it's a type created by the dev, so it's surrounded with [[ ]]
    Generally the name of the type is the name of the file that you need to search for to "decode" it

  • If you're using "Obsidian" and want the text to be a little more **beautiful** then once you pasted the text,
    select all of it then press "SHIFT + TAB", unselect it, then here is it, you have a beautifful **bullet list** =D
